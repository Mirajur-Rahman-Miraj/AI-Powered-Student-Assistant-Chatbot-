# Phase 1 — Domain & Knowledge Definition

## 1. Project Domain

This chatbot is a domain-based AI system focused on:

**Student & Academic Assistance**

It will help students with:
- Academic concept explanations
- Study advice
- Subject recommendations
- Basic greetings and conversation

The chatbot is restricted to academic topics only.

---

## 2. Identified User Intents

The system classifies user inputs into the following intents:

1. Greeting
2. Academic Concept Explanation
3. Study Advice
4. Subject Recommendation
5. Farewell
6. Unknown / Fallback

---

## 3. Example Queries per Intent

### Greeting
- Hi
- Hello
- Hey

### Academic Concept
- What is AI?
- Define OOP
- What is Machine Learning?

### Study Advice
- How to prepare for exams?
- How can I improve my GPA?
- How to study effectively?

### Subject Recommendation
- Which subject should I study first?
- I am weak in math, what should I do?

### Farewell
- Bye
- Goodbye

---

## 4. Knowledge Representation Strategy

The chatbot will:
- Match user keywords
- Map them to an intent
- Select predefined expert response

This follows rule-based expert system architecture.
